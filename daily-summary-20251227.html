üîÑ Query Translation Techniques (Parts 5-9)

Technique Analogy Best For Latency
Multi-Query 3 Different Librarians Vague questions 3x
RAG-Fusion American Idol Voting Precision + Recall 3x+
Decomposition Math Word Problem Complex multi-part 5-10x
Step-Back Professor Context Too-specific questions 2x
HyDE Police Sketch Artist Q-A language mismatch 2x

Key Insights

  • Multi-Query: Generate 3 query variants, search each, combine results for broader coverage
  • RAG-Fusion: Same as Multi-Query but adds Reciprocal Rank Fusion scoring
  • Decomposition: Break complex questions into sub-questions, answer each separately
  • Step-Back: Ask abstract question first to get broader context
  • HyDE: Generate fake answer, search for docs similar to the hypothetical
